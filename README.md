# Bloodveil Client

Download and play Bloodveil RSPS!

## Installation

1. **Download Client**
   - Click **Code** â†’ **Download ZIP**

2. **Download Cache** (Required - One time)

   - Double-click **Run-Bloodveil.bat**

- [Java 11+](https://adoptium.net/) installed
- Windows 7 or higher
- ~400 MB disk space
